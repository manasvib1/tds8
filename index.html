// Calculate CAGR between two values
function cagr(beginValue, endValue, years) {
  return Math.pow(endValue / beginValue, 1 / years) - 1;
}

console.log((cagr(950, 1200, 2) * 100).toFixed(2) + '%'); // outputs CAGR

Financial Formula Examples

Present Value (single cash flow):

\( PV = \dfrac{CF}{(1 + r)^n} \)

Net Present Value (series):

\( NPV = \sum_{t=0}^{N} \dfrac{CF_t}{(1 + r)^t} \)

Weighted Average Cost of Capital (WACC):

\( WACC = \dfrac{E}{V} r_E + \dfrac{D}{V} r_D (1 - T_c) \)
